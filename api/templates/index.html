StackOverflowLiteAPI

StackOverflowLiteApi is a question and answers platform API

Testing only available through Postman.

Create Variables for your requests to avoid typing the long urls

On the POST endpoint for login, go to the "tests" tab and add thie following

const jsonData = JSON.parse(responseBody);
        postman.setEnvironmentVariable("jwt_token", jsonData.access_token);

That {{jwt_token}} variable created is being populated with the access token. Now go to the necessary endpoints endpoints and set up the headers

Key: Authorization
Value: JWT {{jwt_token}}

Endpoints to the API

1. Posting a question

- POST https://zstackoverflowliteapi.herokuapp.com/api/v1/questions

2. Fetch all questions

- GET https://zstackoverflowliteapi.herokuapp.com/api/v1/questions

3. Fetch a specific question

- GET https://zstackoverflowliteapi.herokuapp.com/api/v1/questions/<'int:questionID>

4. Edit a specific question

- PUT https://zstackoverflowliteapi.herokuapp.com/api/v1/questions/<'int:questionID>

5. Delete a specific question

- DELETE https://zstackoverflowliteapi.herokuapp.com/api/v1/questions/<'int:questionID>

6. Posting an answer to the specific question

- POST https://zstackoverflowliteapi.herokuapp.com/api/v1/questions/<'int:questionID>/answers

7. Fetch an answer(s) to the specific question

- GET https://zstackoverflowliteapi.herokuapp.com/api/v1/questions/<'int:questionID>/answers

8. Fetch a specific answer from a specific question

- GET https://zstackoverflowliteapi.herokuapp.com/api/v1/questions/<'int:questionID>/answers<'int:answerID>

9. Edit a specific answer from a specific question

- PUT https://zstackoverflowliteapi.herokuapp.com/api/v1/questions/<'int:questionID>/answers<'int:answerID>

10. Delete a specific answer from a specific question

- DELETE https://zstackoverflowliteapi.herokuapp.com/api/v1/questions/<'int:questionID>/answers<'int:answerID>
